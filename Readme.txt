# CSharp Console RPG (Boss Rush)

Proyecto desarrollado en C# .NET como práctica de arquitectura de software y POO.
Estado actual: **MVP Funcional**

## Descripción
Un juego de rol por turnos donde controlas un grupo de 3 héroes para enfrentarte a jefes con mecánicas únicas.

## Características Técnicas
* **Core:** Sistema de combate por turnos y gestión de estados.
* **Personajes:** Implementación de herencia y polimorfismo para Héroes y Jefes.
* **Interfaz:** Consola pura (CLI).

## Cómo jugar
1. Clonar el repositorio.
2. Abrir `TestProject.sln` en Visual Studio o VS Code.
3. Ejecutar el proyecto (`dotnet run` o F5).
